# Huge Navigation Exercise [![Build Status](https://travis-ci.org/thiagog3/hg-nav-exercise.svg?branch=master)](https://travis-ci.org/thiagog3/hg-nav-exercise) [![Built with Grunt](https://cdn.gruntjs.com/builtwith.png)](http://gruntjs.com/)

## Overview

I followed all recomendations and guideline to do this exercise. All JS code here is completelly vanilla Javascript and I used SASS to make styles. The javascript documentation has generated by jsDoc and is available on ./doc folder. Unit tests has created using Mocha with Chai and Sinon to mock JSON request and Karma execute them. Tests can be run by this following command:

```
$ npm test
```
To run application, just run:

```
$ npm i && npm start
```
